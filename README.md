# Codelabs - Android Kotlin Compose

Este repositorio contiene una colección de proyectos y tutoriales de Android desarrollados con Kotlin y Jetpack Compose.

## Estructura del Repositorio

### 📁 Módulo 1: Affirmations App
- **Ruta**: `modulo1/basic-android-kotlin-compose-training-affirmations-intermediate/`
- **Descripción**: Aplicación intermedia de afirmaciones con lista desplazable de 10 tarjetas

### 📁 Módulo 2: Woof App  
- **Ruta**: `modulo2/basic-android-kotlin-compose-training-woof-main/`
- **Descripción**: Aplicación con fotos de perros incluyendo nombre, edad y actividad favorita

### 📁 Módulo 3: Proyecto Android Completo
- **Ruta**: `modulo3/`
- **Descripción**: Proyecto Android completo con configuración Gradle

### 📁 Módulo 4: Proyecto Android Completo
- **Ruta**: `modulo4/`
- **Descripción**: Segundo proyecto Android completo con configuración Gradle

## Cómo Usar

1. Clona este repositorio:
   ```bash
   git clone https://github.com/isabella041/Codelabs.git
   ```

2. Abre el módulo deseado en Android Studio

3. Ejecuta el proyecto en un emulador o dispositivo
