## 🚀 Cómo Subir Tus Proyectos desde Android Studio

### Paso 1: Prepara tu proyecto en Android Studio
1. Abre tu proyecto en Android Studio
2. Asegúrate de que el proyecto compile y funcione correctamente
3. Guarda todos los cambios (Ctrl + S)

### Paso 2: Copia los archivos del proyecto
1. Navega a la carpeta donde está guardado tu proyecto Android
2. Copia **TODOS** los archivos y carpetas del proyecto:
   - `app/`
   - `build.gradle` o `build.gradle.kts`
   - `settings.gradle` o `settings.gradle.kts`
   - `gradle.properties`
   - `gradlew` y `gradlew.bat`
   - `gradle/`
   - `.gitignore`
   - Cualquier otra carpeta o archivo del proyecto

### Paso 3: Pega en la carpeta correspondiente
1. Ve a `C:\Users\Isabela\CascadeProjects\Codelabs\`
2. Entra en la carpeta del proyecto (ej: `Proyecto1/`)
3. Pega todos los archivos allí

### Paso 4: Sube a GitHub con estos comandos
Abre una terminal en `C:\Users\Isabela\CascadeProjects\Codelabs\` y ejecuta:

```bash
# Agregar los nuevos archivos
git add .

# Hacer commit con mensaje descriptivo
git commit -m "Agregar Proyecto1: [Nombre del proyecto]"

# Subir a GitHub
git push origin main
```
